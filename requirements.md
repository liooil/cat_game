# 像素猫猫盘文玩 - 星露谷深度定制版需求文档

## 1. 核心玩法 (经济与养成)
*   **双螺旋放置**：
    *   **猫咪养成**：领养猫咪，给猫咪挂手串，喂食罐头和三文鱼恢复心情。
    *   **玩家管理 (新增)**：玩家不仅有资金，还有**精力值 (Energy, 0-100)**。
*   **盘包浆机制**：
    *   把手串戴在猫咪脖子上自动盘，或玩家手动盘。
    *   包浆速度受到多个因素综合影响：**猫咪的心情** + **家具 Buff** + **食物 Buff** + **玩家自身的精力值**。
    *   **玩家精力越高，全局盘串（包括手动和猫盘）的速度越快**。
*   **工作与休息 (新增)**：
    *   **去工作**：当资金为 0 时（或任何时候），玩家可以选择“去工作”，一次性获得 500 金币，但会消耗 50 点精力值。
    *   **睡觉**：玩家可以通过“睡觉”选项恢复精力值。
*   **经济循环**：买手串 -> 维持人猫高状态 -> 快速包浆 -> 高价卖出 -> 解锁新房间与更高级的手串。

## 2. 美术风格 (星露谷物语 16-bit 复古风)
> **技术美术（Tech-Artist）代码渲染核心准则：完全使用 Vanilla JS Canvas API，零外部图片。**

*   **画幅**：640 x 480 (4:3) 的复古舒适比例。
*   **1. 调色板与氛围 (Color Palette & Vibe)**
    *   摒弃高饱和度的纯色，采用**自然、低饱和的暖色调**（莫兰迪色系）。草地为带黄的灰绿 (`#7ea258`)，木材为胡桃木色 (`#8b5a2b`)。
    *   **现实时间光照**：通过全局半透明遮罩 (`rgba`) 叠加环境光，营造清晨、白天、黄昏、夜晚的天空和室内氛围变幻。
*   **2. 伪 3D 与阴影 (Pseudo-3D & Drop Shadows)**
    *   **一切皆有阴影**：猫咪、家具、南瓜、手串下方必须绘制半透明黑色椭圆 (`rgba(0,0,0,0.2)`)。
    *   **体积感边缘**：家具（烤箱、橱柜）必须有亮面、暗面，并在底部增加深色线条体现厚度。
*   **3. 像素画纹理细节 (Pixel Texture Details)**
    *   **木板纹理**：通过平行深色细线（偶尔打断）模拟木地板/桌面的拼花缝隙。
    *   **墙纸纹理**：通过极低透明度的垂直/网格线条模拟墙纸质感。
    *   **植物/草地**：使用不同深浅绿色小方块模拟颗粒感。
*   **4. 圆润的像素边缘 (Rounded Pixel Shapes)**
    *   绝对不用死板的直角矩形！使用 `roundRect` 组合，让猫咪、布艺家具看起来圆润、柔软、毛茸茸。猫咪的身体是重叠圆角的拼接。
*   **5. 动画与微小反馈 (Micro-Animations)**
    *   **生动微反馈**：走路时身体平移，**尾巴左右微微摇摆**；交互时产生 `♥` 或 `✨`。
    *   **跳跃过渡**：猫咪跳到家具（沙发、桌子、猫爬架、烤箱）上时，必须有平滑的**抛物线跳跃动画**。

## 3. 场景与互动 (Scenes)
*   **初始小屋 (默认)**：一扇窗户（纯净天空，无怪异斜光），电脑桌（带显示器主机），小沙发。
*   **温馨小屋 (北欧风厨房)**：
    *   格子花纹的墙纸，木纹错落拼花地板。
    *   一个立体的星露谷风**大烤箱**（有玻璃门、旋钮、红火苗、内透暖光）。
    *   一个放着**生动像素大南瓜**的木质橱柜。猫咪可以跳上烤箱。
*   **花园洋房 (欧式三区)**：
    *   一楼：欧式窗户、楼梯、沙发。
    *   二楼：钢琴、大书架、欧式钢笔和黑色墨水。
    *   花园：草地，固定的郁金香（修复抖动），以及一个巨型“猫猫绿植草垛”。
*   **家具互动**：双色猫窝（猫在内睡觉）、大号猫抓板（猫贴近抓挠）、豪华猫爬架（猫跳上平台趴着）。

## 4. 游戏实体细节
*   **猫咪**：7种花色（橘、奶牛、狸花、白、黑、暹罗、三花）。鼠标精准选中。
*   **手串 (立体化与莫兰迪色)**：
    *   所有的串盘在地上或戴在脖子上，必须有**底部阴影、球体内发光、随包浆加深的高光**，立体感十足。
    *   **菩提根特殊定制 (莫兰迪 3 变种)**：每次购买随机抽选——1. 纯色莫兰迪；2. 不同珠子形成整串渐变；3. 单颗珠子内部的绝美渐变。

## 5. UI 与架构要求
*   **UI 布局**：
    *   左侧面板必须有固定高度的容器或使用 `visibility`，彻底杜绝按钮因为文本折叠而集体上下跳动的 Bug。
    *   新增 **【玩家状态】** 面板：显示精力和对应的工作/睡觉按钮。
    *   选中猫咪后，其包浆数值必须**实时、持续跳动更新**（保留两位小数）。
    *   修复戴手串逻辑，确保从背包中扣除。
*   **代码模块化**：严格分为 `index.html`, `style.css`, `js/data.js`, `js/state.js`, `js/cat.js`, `js/renderer.js`, `js/ui.js`, `js/main.js`。完全基于本需求文档重写。
